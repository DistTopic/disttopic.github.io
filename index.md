## What we build

DistTopic develops open source tools that make game development more accessible. Our projects focus on **cross-platform compatibility**, **modern architectures**, and **security-first** releases.

### Highlights

- **Cross-platform** — All our tools run on Windows, macOS, and Linux
- **Verified releases** — Every binary includes SHA-256 checksums for integrity verification
- **Inner source** — Our projects follow inner source best practices with clear contribution guidelines
- **Modern stack** — Built with .NET 10, Avalonia UI, and other modern frameworks

## Featured project

### [Assets And Map Editor](/projects/assets-and-map-editor)

A cross-platform visual editor for Tibia game assets — sprites, items, and outfits.
