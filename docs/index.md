## What we build

<div class="grid cards" markdown>

- :material-monitor-multiple: **Cross-platform**

    All tools run natively on Windows, macOS, and Linux. Single-file executables — no installation required.

- :material-shield-check: **Verified releases**

    Every binary ships with SHA-256 checksums. Integrity verification before you run anything.

- :material-source-repository: **Inner source**

    Projects follow inner source practices: structured contribution guides, GitFlow, and security advisories.

- :material-lightning-bolt: **Modern stack**

    Built with .NET 10 LTS, Avalonia UI 11, C# 13, and GitHub Actions CI/CD with CodeQL scanning.

</div>
