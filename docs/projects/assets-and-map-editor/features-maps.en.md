# OTBM Maps

View and inspect your server map without any additional tools.

![Top-down OTBM map viewer](/assets/images/screenshots/editor-map-view.png)

---

## What you can do

- **Top-down view** — see the full map with terrain types color-coded
- **Smooth zoom and scroll** — navigate from a specific region to a full world overview
- **Inspect tiles** — click any position to see the items, floor, and attributes present
- **Multiple floors** — switch between each map floor with a single click on the selector
- **Mini-map** — quick navigation panel so you never get lost on large maps
