# Sprites & Animations

Browse every sprite in your server right from the editor — no extra tools or manual unpacking needed.

![OTB editor with sprite and animation viewer](/assets/images/screenshots/editor-otb-sprites.png)

---

## What you can do

- **Browse by category** — filter between items, creatures, effects, and projectiles with a single click
- **Zoom and adjustable grid** — zoom into individual sprites or view a full grid at once
- **Animation preview** — play any animated sprite frame by frame
- **Export as PNG** — save individual sprites in full quality directly from the editor
- **Import external sprites** — load PNG files from outside with a live preview before confirming the import
