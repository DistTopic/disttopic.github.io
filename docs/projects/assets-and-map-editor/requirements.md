# Requisitos de Sistema

---

## Mínimo e recomendado

| Recurso | Mínimo | Recomendado |
|---------|--------|-------------|
| RAM | 512 MB | 2 GB |
| Espaço em disco | 50 MB | 200 MB |
| Windows | 10 (1903+) | 11 |
| macOS | 12 Monterey | 13 Ventura+ |
| Linux | Ubuntu 20.04 / Debian 11 | Ubuntu 22.04+ |

---

## Plataformas disponíveis

O editor é distribuído como executável autônomo — sem instalação, sem dependências externas.

| Sistema operacional | Arquitetura |
|--------------------|-------------|
| Windows | x64 · ARM64 |
| macOS | Intel (x64) · Apple Silicon (ARM64) |
| Linux | x64 · ARM64 |

!!! tip "Sem instalação necessária"
    Basta baixar e executar. Nenhuma versão do .NET precisa estar instalada no sistema.

Veja a [página de releases](https://github.com/DistTopic/assets-and-map-editor/releases) para todos os downloads disponíveis.
