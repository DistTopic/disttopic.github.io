## What is it?

The **Assets And Map Editor** is a desktop application for viewing and editing Tibia game data files. Built for Open Tibia servers, it supports **DAT**, **SPR**, **OTB**, and **OTBM** formats with a modern, cross-platform interface.

## Supported platforms

| Platform | Architecture | Format |
|----------|-------------|--------|
| Windows | x64, ARM64 | `.zip` (portable) |
| macOS | x64 (Intel), ARM64 (Apple Silicon) | `.zip` (app bundle) |
| Linux | x64, ARM64 | `.zip` (portable) |
