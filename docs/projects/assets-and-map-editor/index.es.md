## ¿Qué es?

El **Assets And Map Editor** es una aplicación de escritorio para visualizar y editar archivos de datos de Tibia. Soporta los formatos **DAT**, **SPR**, **OTB** y **OTBM** con una interfaz moderna y multiplataforma.

## Plataformas soportadas

| Plataforma | Arquitectura | Formato |
|------------|-------------|---------|
| Windows | x64, ARM64 | `.zip` (portátil) |
| macOS | x64 (Intel), ARM64 (Apple Silicon) | `.zip` (app bundle) |
| Linux | x64, ARM64 | `.zip` (portátil) |
