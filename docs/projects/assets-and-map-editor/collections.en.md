## Creating a collection

1. In the side palette, click the **Collection** tab
2. Name your collection in the text field
3. Add items via the context menu (right-click any item in the list)

<!-- TODO: capture screenshot of context menu showing "Add to Collection" and "Add to Sub-Collection" options -->
![Context menu with collection options](/assets/images/screenshots/v2-collection-context-menu.png)

### Context menu options

| Option | What it does |
|--------|-------------|
| **Add to Collection** | Adds the item to the currently selected root collection |
| **Add to Sub-Collection** | Creates or adds to a sub-collection within the current collection |

## Navigating collections

<!-- TODO: capture screenshot showing navigation between main collection and sub-collection, with updated header -->
![Navigation between collection and sub-collection](/assets/images/screenshots/v2-collection-navigation.png)

- The tab displays the active collection name in the header
- Switch between collections using the navigation controls
- The items list updates immediately when switching collections

!!! tip "Tip"
    Use sub-collections to organize items by type — for example, a "Nature" collection with "Trees", "Flowers", and "Rocks" sub-collections.

## Category-aware transplant

When performing transplant or merge between sessions, you can now filter by category.

<!-- TODO: capture screenshot of merge menu showing per-category options (Items, Outfits, Effects, Missiles) -->
![Merge menu with per-category options](/assets/images/screenshots/v2-merge-categories.png)

| Category | What it includes |
|----------|-----------------|
| Items | All client items (terrain, equipment, decorations) |
| Outfits | Character and mount visuals |
| Effects | Visual effects (magic, impact, etc.) |
| Missiles | Projectiles and trajectory effects |

### Animated previews
