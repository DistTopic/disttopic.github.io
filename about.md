# About DistTopic
{: .fs-8 }

We are an open source organization building tools for the Tibia game development community.
{: .fs-5 .fw-300 }

---

## Mission

DistTopic exists to create **high-quality, cross-platform, open source tools** that empower game developers and server administrators in the Tibia community.

We believe in:

- **Transparency** — All source code is public and auditable
- **Security** — Verified builds with SHA-256 checksums, security scanning via CodeQL
- **Accessibility** — Tools that run on every major operating system
- **Community** — Inner source practices that welcome contributors at every skill level

## Technology

Our projects use modern, well-supported technologies:

- **.NET 10 LTS** — Long-term support runtime
- **Avalonia UI 11** — Cross-platform UI framework
- **GitHub Actions** — CI/CD for builds, testing, and security scanning
- **CodeQL** — Automated vulnerability detection
